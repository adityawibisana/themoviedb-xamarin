## Description:
Xamarin Forms application to show list of movies, using TMDB API. Features:
1. Infinite scroll
2. Display Genre list
3. Show list of movies based on genres
4. Show trailer from Youtube
5. Show user review

## Key Technologies:
1. Xamarin
2. MVVM Pattern
3. Dependency Injection
4. Xamarin: Behavior
5. Data Binding
